npm module for using the aqm0802a with node.js on the raspberry pi

This module is not yet ready for use.

The aqm0802a is a small 8x2 LCD display module that can be connected to the raspberry pi on the i2c bus. You can buy a ready-to-use, no soldering or breadboard needed module that uses the aqm0802a, to use with the raspberry pi 2 from switch science

I will use this page to document things that I learn about this module as I experiment.
