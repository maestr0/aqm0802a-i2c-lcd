aqm0802a-i2c-lcd

Use the aqm0802a 8x2 LCD display in your node.js projects on the raspberry pi

Download .zip Download .tar.gz View on GitHub

Buy the LCD display breakout module

The aqm0802a is a small 8x2 LCD display module that can be connected to the raspberry pi on the i2c bus. You can buy a ready-to-use, no soldering or breadboard needed module that uses the aqm0802a, to use with the raspberry pi 2 from switch science


Read the docs

The datasheet for the module is in Japanese. You can read some English documentation here.
